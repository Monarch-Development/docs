## Innovations
- Save time in consumables configuration
- Have everything under control in a single file (ox_inventory items.lua)
- No configuration needed in the script
- The item function is triggered with a simple server-side export that determines the category and function of the item
- Added player alteration effect in item configuration with preset effects
