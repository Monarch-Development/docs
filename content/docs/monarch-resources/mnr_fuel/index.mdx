Monarch Fuel is the union of cdn-fuel and some optimized parts of ox_fuel plus some improvements made by Monarch Development, the aim of the project is to offer the community a fuel system as feature-rich as cdn-fuel without sacrificing optimization and lean code like ox_fuel.

<GithubInfo owner="Monarch-Development" repo="mnr_fuel" className="border border-purple-600 rounded-md p-4 bg-gradient-to-r from-purple-600 to-transparent no-underline"/>

## Features
- Menu and zones from ox_lib
- Statebags for fuel level
- Nozzle and rope for fuel pumps
- Jerrycans integration for ox_inventory and qb-inventory
