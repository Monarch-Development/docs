## Steps

<Steps>

  </Step>

  Extract the .zip file

  </Step>

  </Step>

## Dependencies and Start Order

```mdx title="server.cfg"
ensure ox_lib
ensure (FRAMEWORK)
ensure (TARGET)
ensure (INVENTORY)
ensure mnr_fuel
```

<Callout title="WARNING" type="warn">
  The `(FRAMEWORK)`, `(TARGET)` and `(INVENTORY)` parts are just indicative, replace them with the names of the resources you use.
</Callout>
