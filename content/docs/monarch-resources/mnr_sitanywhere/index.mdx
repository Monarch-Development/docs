Monarch Sit Anywhere is an optimized and simple resource that makes benches and chairs interactable and it makes it possible to sit on them.

/>

## Features
- [NEW] Game coordinates are converted based on the heading of the object for simple and multiple offsets
- [NEW] Multiple seating positions for chairs and benches
- [NEW] Network system that makes possible for other players to recognize if a seat is occupied
- [NEW] Support for custom chairs and benches that are cloned to force networking of them
